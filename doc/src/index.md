# TEM

A mod for Tron: Evolution.

![tem.gif](./images/tem.gif)

## Features

- Removal of Forced Window Minimize
- Superuser
	- Unlimited Health
	- Unlimited Energy
	- High Damage Multiplier
- RGB Suit
- Anti-Anti-Debugger
- HUD
	- Timer
	- Position
	- Angle
	- Velocity
	- Health
	- In-Game Inputs
- Level Selector

## Limitations

- International retail version 1.01 only
- Offline only
